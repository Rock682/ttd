TTD Auto Fill Pro - Installation Guide

1. Download the ZIP from AI Studio (Settings -> Export to ZIP).
2. Extract the ZIP file on your computer.
3. Open Google Chrome and go to: chrome://extensions/
4. Turn ON "Developer mode" (top right).
5. Click "Load unpacked" (top left).
6. SELECT THE MAIN FOLDER you just extracted (the one that contains manifest.json).

If you see "Manifest file is missing", make sure you are selecting the folder that has the 'manifest.json' file inside it.
In this version, I have moved the manifest.json to the top-level folder to make it easier!
